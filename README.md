# Practica: Mi primer servidor

Este proyecto consiste en la creación progresiva de un servidor en Node.js desarrollado en distintas etapas.  
Se implementa desde un programa básico "Hola Mundo" hasta un servidor HTTP que lee y envía archivos de manera síncrona y asíncrona.  
Finalmente, el proyecto es dockerizado para su ejecución en contenedores.
